# Loops Worksheet

## Example Code

```python
for x in range(0,6,2):
    print(x)

x = 0
while(x<10):
    print(x)
    x+=2

collectives = ['apple', 'Phone', 'book','pen']

for x in collectives:
    if(x[0].lower() == 'p'):
        print(x)

mini_alphabet = [['a','b','c'],
                ['d','e','f'],
                ['g','h','i']]
a = 0
for x in mini_alphabet:
    for y in x:
        if(a%2 == 0):
            print(y)
        a+=1
```
